Creating detection catalog with surface brightness¶

In order to get the surface brightness information, we match the balrog detection catalog (all true objects) with the supernovae and cosmos fields from where the injected objects were taken (matching on the "true_id" columns, which corresponds to the coadd_object_id of the parent object).

As function of magnitude¶

In [21]:
selection_functions = {}
for band in griz:
    grouped = balrog_with_sb.groupby("MAG_{}_BIN".format(band))["DETECTED"]
    nm = grouped.sum()
    nt = grouped.count()
    selection_function = pd.DataFrame(grouped.sum() / grouped.count())
    selection_function = selection_function.rename({"DETECTED": "SELECTION FUNCTION"}, axis=1)
    selection_function["SELECTION FUNCTION ERR"] = selection_function["SELECTION FUNCTION"] * np.sqrt(1./nm - 1./nt)
    selection_function["SELECTION FUNCTION LOW"] = selection_function["SELECTION FUNCTION"] - selection_function["SELECTION FUNCTION ERR"]
    selection_function["SELECTION FUNCTION HIGH"] = selection_function["SELECTION FUNCTION"] + selection_function["SELECTION FUNCTION ERR"]
    selection_function["MAG"] = selection_function.index.get_level_values(0).map(lambda x: x.left).astype(float)
    selection_function["BAND"] = band
    selection_functions[band] = selection_function
